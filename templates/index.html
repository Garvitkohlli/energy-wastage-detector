How It Works

  1. Go to Dashboard - Select an appliance to monitor
  2. Start Auto-Monitoring - System generates random readings every 2 seconds
  3. ML Analyzes Patterns - Learns usage patterns by hour of day
  4. Get Recommendations - System tells you the BEST TIME to cut power
  5. Minimize Impact - Cut power when appliance is least used
